# Expense Tracker
This project is an Expense Tracker Application developed using Spring Boot for the backend and React for the frontend.

## Requirements
- Java: 17
- Spring Boot: 3.x
- Maven
- React + TypeScript with Vite

## Running Locally
Use Docker Compose to run the application locally:

```bash
docker-compose up
```

## Deploying to Kubernetes
To deploy the application to Kubernetes, use the following command:

```bash
kubectl apply -f k8s/
```
